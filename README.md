# **Mi Primera Página**
Este repositorio contiene el desarrollo del taller diagnóstico de Introducción a las Aplicaciones Web, en el cual se aplican conceptos básicos de HTML, CSS y JavaScript.
El objetivo principal es demostrar conocimientos previos sobre el desarrollo web y el uso de herramientas básicas.

***

## Parte Teórica

### 1. ¿Qué es HTML y cuál es su función en la web?
#### **HTML** es un lenguaje de marcado que se utiliza para crear la estructura de una página web. Permite organizar el contenido como textos, imágenes, enlaces y otros elementos que se muestran en el navegador.

---

### 2. ¿Qué es una etiqueta HTML y menciona las más comunes?

#### Una **etiqueta HTML** sirve para definir los elementos que componen una página web.

Algunas de las etiquetas más comunes son:

- html
- head
- body
- title
- h1

---

### 3. ¿Qué es un atributo de una etiqueta HTML y menciona los más comunes?

#### Un **atributo** es una característica adicional que se le asigna a una etiqueta HTML para proporcionar más información o modificar su comportamiento.

Algunos atributos comunes son:

- id
- class
- src
- href
- alt

---

### 4. ¿Qué es CSS y cómo se utiliza para el diseño web?

#### **CSS** es un lenguaje que se utiliza para dar estilo a las páginas web, permitiendo modificar colores, tamaños de letra, posiciones y la apariencia general del contenido.

---

### 5. ¿Qué es una propiedad en CSS y menciona las más comunes?

#### **Una propiedad en CSS** indica qué aspecto visual se va a modificar en un elemento HTML.

Algunas propiedades comunes son:

- color
- background-color
- font-size
- margin
- padding

---

### 6. ¿Qué es un selector en CSS y cuáles tipos existen?

#### Un **selector** se utiliza para indicar a qué elementos HTML se les aplicará un estilo.

Los tipos de selectores más comunes son:

- Selector por etiqueta
- Selector por clase
- Selector por id

---

### 7. ¿Qué es JavaScript y cómo añade interactividad a las páginas web?

#### **JavaScript** es un lenguaje de programación que permite agregar interactividad a las páginas web, como responder a eventos, mostrar mensajes o modificar el contenido sin recargar la página.

---

### 8. ¿Cuáles son los tipos de datos primitivos en JavaScript?

#### Los tipos de datos primitivos en JavaScript son:

- string
- number
- boolean
- null
- symbol

---

### 9. ¿Cómo funcionan las estructuras de control de flujo?

#### Las estructuras de control permiten tomar decisiones y repetir acciones dentro de un programa.

Por ejemplo:

- if
- else
- switch
- for
- while

---

### 10. ¿Por qué es importante usar nombres significativos para variables y métodos?

#### Porque usar nombres claros facilita la lectura del código y ayuda a entender mejor qué función cumple cada variable o método.

---

### 11. ¿Qué es una variable de entorno y por qué son importantes?

#### Una **variable de entorno** es un valor que se utiliza para configurar una aplicación sin necesidad de escribirlo directamente en el código, lo cual ayuda a proteger información sensible.

---

### 12. ¿Qué son las herramientas de desarrollo de Chrome y cómo se accede a ellas?

#### Son herramientas integradas en el navegador que permiten inspeccionar y depurar páginas web.

Se puede acceder a ellas de las siguientes formas:

- Presionando **F12**
- Haciendo click derecho y seleccionar **inspeccionar**

---

### 13. ¿Qué se puede hacer en el panel "Elements"?

#### En el panel **Elements** se puede visualizar y modificar la estructura HTML y los estilos CSS de una página en tiempo real.

---

### 14. ¿Cómo se utiliza el panel "Console" y para qué es útil?

#### El panel **Console** se utiliza para mostrar mensajes, errores y probar código JavaScript mediante instrucciones como console.log().

---

### 15. ¿Qué información se puede obtener del panel "Network" y por qué es importante?

#### El panel **Network** permite ver las peticiones de red, los archivos que se cargan y los tiempos de respuesta, lo cual es importante para analizar el rendimiento de una página web.

---
